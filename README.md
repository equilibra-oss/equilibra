## ūüĒļ The Problem

Student life rests on a **fragile triangle**:

```
          ūüéď Academic Performance
                    /\
                   /  \
                  /    \
                 /      \
                /________\
    ūüí™ Physical          ūüß† Mental
       Health              Balance
```

Neglect just one corner ‚ÄĒ and the whole triangle collapses.  
Most students navigate this alone, without tools to find balance.

**Equilibra changes that.**

## ūüĆü Vision

> *"A thriving student isn't the one who sacrifices sleep for grades, nor the one who forgets their studies to take care of themselves. It's the one who holds all three corners at once."*

Equilibra is an open source platform built **by students, for students** ‚ÄĒ to monitor, understand, and improve the three pillars of student life in a truly holistic way.

### ūüí™ Physical Health

### ūüß† Mental Balance

## ūüöÄ Getting Started

```bash

cd equilibra

# Install dependencies

# Set up environment variables
cp .env.example .env
```
